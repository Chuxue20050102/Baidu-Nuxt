# Baidu-Nuxt

本项目是一个基于 Nuxt3 框架仿写的百度首页，采用 Vue3 组合式 API、组件化开发、全局主题联动等前端技术。

## 主要功能
- 仿百度首页布局与交互
- 组件化开发（导航栏、搜索框、热搜榜、辅助模式、登录弹窗、FooterBar等）
- 辅助模式/主题色切换，配色全局联动
- 响应式布局
- 全局状态管理（useState/composables）



## 如何运行

1. **安装依赖**

```bash
npm install
```

2. **启动开发服务器**

```bash
npm run dev
```

3. **打包构建**

```bash
npm run build
```

4. **预览生产环境**

```bash
npm run preview
```
